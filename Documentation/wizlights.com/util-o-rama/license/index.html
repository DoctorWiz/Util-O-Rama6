Open Source:

The Util-O-Rama suite is open source under a GPL 3 license. The repository is here.
Util-O-Rama is written in C# v10.0, .Net 6.0, and WinForms using Visual Studio 2022 Community Edition which you can get for free here.
You may be able to use Visual Studio Code instead, I’m not sure, I haven’t tried it since Visual Studio Community is free. You can download Visual Studio Code here.
Util-O-Rama uses several of the SyncFusion controls for WinForms. They offer a free ‘community’ license for open source projects. You can download it here and get a license here.
I use quite a few extensions for Visual Studio.  None of them are required to compile Util-O-Rama, nor to look at and modify the code.  However, installing some, most, or all of them will certainly help if you wish to work on the code.  Contact me for a list.  Likewise, if you want to join the development team please be sure to contact me.
