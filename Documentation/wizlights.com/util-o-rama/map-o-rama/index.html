Map-O-Rama

Version Alpha 4.6.0.2α overview

Util-O-Rama is a suite of utilities written to manipulate sequence files for Light-O-Rama Showtime version 4.x and xLights.

Map-O-Rama: Is the first utility I wrote in the Util-O-Rama suite.  As you probably already know, Light-O-Rama Showtime version 4.x saves a copy of your channel configuration with each and every sequence.  There is a function to export and import the channel configuration but that only works when you keep the same number of channels in the same order.  If you add a channel in the middle somewhere, or rearrange some of them, importing the other channel configuration screws everything up royally.  The original intent of Map-O-Rama was to apply new channel configurations from a master to all my sequences. So say you do add a couple channels, or rename some, or re-arrange them in the tracks. You can now "map" your new layout to all of your sequences!  I quickly figured out that this also works really well for remapping the channels from a shared or downloaded sequence which may have a vastly different layout.  And multiply that usefulness if you are importing multiple sequences from the same source.  Map-O-Rama is pretty versatile about remapping channels so you may find other uses, be sure to share them.

Start by selecting your new "master" sequence on the right half of the window.  Select an old, no longer correct sequence on the left half.  If you only have minor changes, the AutoMap button will match up channels by name.  You can then manually map the changes by selecting corresponding channels on each side and clicking the Map button.  Finally, save the new sequence.  It will take all the effects from the old sequence on the left and apply them to the new master channel layout on the right!  If you have multiple old sequences to update, you can save the map to a file to be reapplied to the other sequences.

Likewise, if you are importing someone elses sequences you downloaded, you would first select your "master" sequence on the right, select the new downloaded one on the left, and then manually map channels across as you see fit.  Also note that you can map a channel on the source side to more than one channel on the master (destination) side.  If you want to do multiple sequences downloaded from the same author, be sure to save the map and then the rest of them will be a breeze!

If you've ever used the import mapping in xLights, this is very similar.

Map-O-Rama also includes "fuzzy" name matching, so any channels names that do not have an exact match will get matched to any leftover channels where the name is very similar.  If you prefer your master on the left, there is an option to switch.  You can also view the mappings in a list and sort them by name alphabetically, or by the order they appear in, or by unit and channel number or universe and DMX channel.
