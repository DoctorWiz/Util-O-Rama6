Vamp-O-Rama uses the Vamp Plugins to generate timing marks and channels for both Light-O-Rama Showtime version S4 and xLights.  It also creates a Showtime sequence with a bunch of channels (which can then be imported into xLights) based on the vamp results.  Vamp-O-Rama performs extensive post processing on the vamp results to align, scale, and label the results to make them much more useful in Showtime and/or xLights.  Vamp-O-Rama can do this with all the Vamp plugins, not just the Queen Mary ones.  Vamp-O-Rama can create not only beat tracks, but tracks for bars ùÑÅ, whole notes ùÖù, half notes ùÖû, quarter notes ùÖü, eighth notes ùÖ†, and sixteenth notes ùÖ°.  It can handle 4/4 time or 3/4 time.  It not only performs polyphonic transcription, but creates individual channels with on-off or ramp effects for each note.  It creates analog channels for things like tempo, pitch, key, and chords.  It can create full spectrograms with each frequency in its own channel.

Prerequisites:  You must first download and install the 64-bit version of the Vamp Plugins which can be downloaded here.  You must also download the 64-bit version of Sonic Annotator which can be downloaded here.  Both Windows installers (.msi) and portable versions are available for each product.  Installing them is the same as installing any other piece of software so I'm not going into details here.  Likewise, to use Vamp-O-Rama you must first download and install the Util-O-Rama suite.  Instructions are here.  This help file assumes you have already installed the Util-O-Rama suite.
